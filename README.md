# Technologies Used
* Frontend: HTML, CSS, Bootstrap 5, javascript
* Backend: PHP, Laravel
* Database: Mysql
# Steps to Run The project

Commands : 
* composer install
* Php artisan key:generate
* cp .env.example .env
* php artisan migrate:fresh
* php artisan db:seed
* php artisan serve
* php artisan optimize:clear (optional :  Laravel's cache)
